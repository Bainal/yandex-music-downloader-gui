# Yandex Music Downloader — GUI
Графическая версия загрузчика музыки для Яндекс.Музыки. Вставьте ссылку, выберите папку и нажмите «Скачать».

<img width="674" height="586" alt="изображение" src="https://github.com/user-attachments/assets/bdfe6fe2-bbc6-42ca-bd7d-d66c791e87cd" />


## Возможности
- Скачивание по ссылкам: артист / альбом / трек / плейлист / «Мне нравится».
- Выбор качества (AAC / FLAC).
- Выбор папки сохранения.
- Многопоточная загрузка (1–4 потока).
- Предпросмотр, что будет загружено.
- Кнопка «Открыть папку».
- Сохранение токена по желанию (через галочку).

## Требования
- Python 3.9+
- Доступ в интернет

## Установка (локально из репозитория)
```powershell
py -3.14 -m pip install -e .
```

## Запуск GUI
```powershell
yandex-music-downloader-gui
```

## Как пользоваться
1. Вставьте ссылку на Яндекс.Музыку.
2. Укажите папку для скачивания.
3. Выберите качество и количество потоков.
4. Вставьте токен и отметьте «Сохранить» (если хотите запоминать его).
5. Нажмите «Скачать».

### Примеры ссылок
```text
https://music.yandex.ru/artist/208167
https://music.yandex.ru/album/294912
https://music.yandex.ru/album/11644078/track/6705392
https://music.yandex.ru/users/<user>/playlists/3
https://music.yandex.ru/playlists/lk.<uuid>
```

## Токен
Токен нужен для доступа к вашему аккаунту (включая «Мне нравится»).
Инструкция по получению токена:
```
https://yandex-music.readthedocs.io/en/main/token.html
```

Если включена галочка «Сохранить», токен сохраняется в конфиг и подгружается при следующем запуске:
- Windows: `%APPDATA%\yandex-music-downloader\gui.json`
- Linux: `~/.config/yandex-music-downloader/gui.json`
